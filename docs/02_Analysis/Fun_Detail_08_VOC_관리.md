# 8. VOC 관리
**설명**: 오더별 고객 불만·문의(Voice of Customer)를 접수·처리하는 기능.
**Phase**: Phase 3

## 8.1 VOC 등록
**상세**: 오더를 선택하여 불만·문의 내용을 등록. 등록 후 초기 처리상태는 OPEN. 담당자에게 알림 발송.
**권한**: 회원
**Phase**: Phase 3

## 8.2 VOC 답변
**상세**: 접수된 VOC에 담당자가 답변 등록. 답변 완료 시 처리상태를 IN_PROGRESS 또는 CLOSED로 변경. 답변 등록 후 고객에게 알림 발송.
**권한**: 운영자/관리자
**Phase**: Phase 3

## 8.3 VOC 조회
**권한**: 회원/운영자/관리자
**Phase**: Phase 3
