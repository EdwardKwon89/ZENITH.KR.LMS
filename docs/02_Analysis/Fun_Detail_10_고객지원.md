# 10. 고객지원
**설명**: 회원이 서비스 이용 중 발생하는 문의·공지 정보를 제공하는 기능.
**Phase**: Phase 4

## 10.1 QnA
**상세**: 회원이 1:1 문의를 등록하고 관리자가 답변하는 기능. 문의 등록 후 처리상태(접수·처리중·완료) 확인 가능.
**권한**: 회원/관리자
**Phase**: Phase 4

## 10.2 FAQ
**상세**: 자주 묻는 질문과 답변을 카테고리별로 관리. 관리자가 FAQ를 등록·수정·삭제하며 회원은 조회만 가능.
**권한**: 회원/관리자
**Phase**: Phase 4
