# 2. 회원관리
**설명**: 개인회원·법인회원·관리자 계정의 등록, 수정, 조회, 탈퇴 등 회원 전반을 관리하는 기능.
**Phase**: Phase 1

## 2.1 개인회원
**설명**: B2C 개인 사용자의 회원가입부터 정보관리·탈퇴까지 전 생애주기를 관리.
**Phase**: Phase 1

### 2.1.1 회원가입
**설명**: 개인회원이 서비스 이용을 위해 계정을 생성하는 절차.
**Phase**: Phase 1

#### 2.1.1.1 본인확인
**상세**: 이름·생년월일·휴대폰번호를 입력하고 SMS 인증번호로 본인 확인. 중복 가입 방지를 위해 동일 휴대폰번호로 기존 계정 존재 여부 체크.
**권한**: 개인회원
**Phase**: Phase 1

#### 2.1.1.2 개인정보 입력
**상세**: 본인확인 완료 후 로그인ID·비밀번호·이메일·주소를 입력하여 계정 생성. 로그인ID와 이메일은 중복 불가. 주소는 국가코드+우편번호+주소+상세주소로 구성.
**권한**: 개인회원
**Phase**: Phase 1

#### 2.1.1.3 소셜 로그인 연동 가입
**상세**: 소셜 계정(Google 등)으로 최초 접근 시 소셜 인증 후 추가 정보(이름·주소·이메일)를 입력하여 계정 생성. 이후 소셜 로그인으로 자동 인증.
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.2 회원정보 수정
**상세**: 이름·생년월일·휴대폰번호·주소·이메일 주소 수정 가능. 로그인ID는 변경 불가. 수정 완료 후 최종수정일시 갱신.
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.3 회원 등급 조회
**상세**: 이용 실적(발송 횟수, 누적 금액)에 따라 부여된 등급(Family/Bronze/Silver/Gold/Platinum) 확인. 등급별 운송비 할인율(0%~10%) 및 다음 등급까지의 남은 실적 정보를 함께 표시.
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.4 잔액 충전
**상세**: 선불 방식으로 서비스 이용 잔액을 충전. 충전 금액은 청구서 결제에 사용 가능. 충전 후 잔액 및 충전 이력을 확인할 수 있음.
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.5 잔액 환불
**상세**: 충전 잔액 중 미사용 금액에 대해 환불 요청. 환불 처리는 관리자 검토 후 진행되며 청구서에 반영.
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.6 청구내역 조회
**상세**: 내 계정에 발행된 청구서 목록과 항목별 청구 이력 조회. 기간·운송수단별 필터링 가능하며 청구서 PDF 출력 지원.
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.7 회원탈퇴
**상세**: 본인이 직접 탈퇴 신청. 개인정보(이름·생년월일·연락처·주소·이메일)는 Null 처리하고 로그인ID만 보존(Soft Delete). 탈퇴 후 동일 ID로 재가입 불가. (비고: Soft Delete 처리)
**권한**: 개인회원
**Phase**: Phase 1

### 2.1.8 개인정보 활용동의
**상세**: 서비스 이용 약관 및 개인정보 처리방침 동의 여부 관리. 필수 동의 항목 미동의 시 서비스 이용 제한.
**권한**: 개인회원
**Phase**: Phase 1

## 2.2 법인회원
**설명**: B2B 법인 고객의 가입부터 조직(부서)·관리자 관리, 탈퇴까지 처리.
**Phase**: Phase 1

### 2.2.1 법인 회원가입
**설명**: 법인 계정 생성 절차. 가입 후 관리자 심사 승인 시 계정 활성화.
**Phase**: Phase 1

#### 2.2.1.1 법인관리자 본인확인
**상세**: 법인관리자의 이름·생년월일·휴대폰번호를 입력하고 SMS 인증번호로 본인 확인.
**권한**: 법인관리자
**Phase**: Phase 1

#### 2.2.1.2 법인정보 입력
**상세**: 법인명·대표자명·사업자등록번호·대표전화·이메일·주소 입력. 사업자등록번호 중복 체크.
**권한**: 법인관리자
**Phase**: Phase 1

#### 2.2.1.3 첨부파일 업로드
**상세**: 사업자등록증 등 심사에 필요한 증빙 서류 파일 업로드. 파일은 MinIO에 저장되며 관리자가 열람하여 승인·거부 결정. (비고: 허용 파일 PDF·JPG·PNG)
**권한**: 법인관리자
**Phase**: Phase 1

#### 2.2.1.4 부서정보 입력
**상세**: 가입과 함께 최초 부서명 및 부서관리자 정보를 입력. 부서 추가는 가입 승인 후 별도 관리 메뉴에서 처리.
**권한**: 법인관리자
**Phase**: Phase 1

### 2.2.2 법인정보 수정
**상세**: 법인 대표자명·법인주소·대표전화·이메일 수정 가능. 법인명·사업자번호는 변경 불가(변경 필요 시 관리자 문의).
**권한**: 법인관리자
**Phase**: Phase 1

### 2.2.3 법인관리자 정보수정
**상세**: 법인관리자 본인의 이름·생년월일·휴대폰번호·주소·이메일 수정.
**권한**: 법인관리자
**Phase**: Phase 1

### 2.2.4 부서 관리
**상세**: 법인 소속 부서를 추가·수정·삭제. 삭제 시 부서ID만 보존하고 부서명 등 정보는 Null 처리(Soft Delete).
**권한**: 법인관리자
**Phase**: Phase 1

### 2.2.5 부서관리자 정보수정
**상세**: 부서관리자 본인의 이름·생년월일·휴대폰번호·주소·이메일 수정.
**권한**: 부서관리자
**Phase**: Phase 1

### 2.2.6 법인 회원탈퇴
**상세**: 법인관리자가 탈퇴 시 법인·법인관리자·부서·부서관리자 전체 Soft Delete 처리. 법인ID·법인관리자ID·부서ID·부서관리자ID만 보존하고 개인정보는 전부 Null 처리. 법인증빙파일은 완전 삭제.
**권한**: 법인관리자
**Phase**: Phase 1

## 2.3 관리자
**설명**: 시스템 관리자가 회원 계정 및 운송비용 정책을 관리하는 기능.
**Phase**: Phase 1

### 2.3.1 개인회원 관리
**상세**: 전체 개인회원 목록 조회 및 회원별 등급·할인율 관리. 각 등급(Family~Platinum)별 할인율과 영업이익율을 개별 또는 일괄 설정 가능. 회원의 누적 실적(산정 기간 기준)을 직관적으로 확인.
**권한**: 관리자
**Phase**: Phase 1

### 2.3.2 법인 심사
**설명**: 법인 회원가입 신청 건에 대해 증빙서류를 확인하고 승인 또는 거부하는 절차.
**Phase**: Phase 1

#### 2.3.2.1 첨부파일 확인
**상세**: 가입 시 업로드된 사업자등록증 등 증빙파일을 MinIO에서 불러와 관리자가 열람.
**권한**: 관리자
**Phase**: Phase 1

#### 2.3.2.2 승인 처리
**상세**: 심사 통과 시 법인 계정 상태를 ACTIVE로 변경하고 법인관리자에게 승인 안내 알림 발송.
**권한**: 관리자
**Phase**: Phase 1

#### 2.3.2.3 거부 처리
**상세**: 심사 거부 시 approval_status를 REJECTED로 변경하고 거부 사유를 법인관리자에게 알림. 거부 후 재신청 가능.
**권한**: 관리자
**Phase**: Phase 1

### 2.3.3 운송비용 관리
**설명**: 회원별 서비스 운송비 할인율 및 영업이익율을 등록·수정하는 기능.
**Phase**: Phase 1

#### 2.3.3.1 개인회원 운송비 등록/수정
**상세**: 개인회원별로 AIR·SEA·CIR·CCL 서비스에 대한 할인율(%)과 영업이익율(%) 설정. 적용 시작일과 종료일을 지정하여 기간 관리 가능.
**권한**: 관리자
**Phase**: Phase 1

#### 2.3.3.2 법인회원 운송비 등록/수정
**권한**: 관리자
**Phase**: Phase 1

### 2.3.4 승급 대상자 심사/관리 [신규]
**설명**: 시스템 설정(`GRADE_EVAL_PERIOD`)에 정의된 산정 기간 동안의 실적을 바탕으로 승급 요건을 충족한 회원 리스트를 관리자가 최종 승인.
**상세**:
1. **대상자 조회**: 발송 횟수 및 누적 금액 조건을 모두 충족하여 `PROMOTABLE` 상태인 회원 리스트 확인.
2. **실적 검증**: 관리자가 해당 회원의 상세 오더 이력을 검토.
3. **승인/반려**: 승인 시 즉시 목표 등급으로 승격되며 할인율이 자동 적용됨. 반려 시 신청 상태가 해제되며 사유 기록.
4. **이력 관리**: 등급 변경 일시, 변경 전/후 등급, 승인자 정보를 `grade_promotion_history`에 기록.
**권한**: 관리자
**Phase**: Phase 1
