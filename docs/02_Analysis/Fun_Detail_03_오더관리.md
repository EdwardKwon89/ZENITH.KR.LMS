# 3. 오더관리
**설명**: 화물 운송 의뢰 단위인 오더를 등록·관리하는 핵심 기능. 오더는 송하인·수하인·화물·서비스 정보로 구성.
**Phase**: Phase 2

## 3.1 오더 기본관리
**설명**: 오더의 신규 등록, 수정, 삭제, 조회를 처리.
**Phase**: Phase 2

### 3.1.1 오더 등록
**설명**: 새 운송 오더를 시스템에 등록. 오더번호는 시스템이 자동 생성(예: ORD-20260415-00001). 초기 상태는 REGISTERED.
**Phase**: Phase 2

#### 3.1.1.1 송하인 정보 입력
**상세**: 화물을 보내는 송하인(발송자) 정보 입력. 이름·국가코드·우편번호·주소·상세주소·전화번호. 법인 송하인인 경우 사업자등록번호 추가 입력.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

#### 3.1.1.2 수하인 정보 입력
**상세**: 화물을 받는 수하인(수취인·수입자) 정보 입력. 이름·국가코드·우편번호·주소·상세주소·전화번호. 법인 수하인인 경우 사업자등록번호 추가 입력.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

#### 3.1.1.3 화물 정보 입력
**상세**: 운송 화물의 상세 정보 입력. 품명·HS코드·원산지(국가코드)·개별수량(PCS)·포장수량(CTNS/PKGS)·총중량(kg)·순중량(kg)·용적(CBM). 운송원가 계산에 총중량과 CBM이 사용됨.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

#### 3.1.1.4 특수화물 기재
**상세**: 일반화물 외 특수 취급이 필요한 항목 기재. ① 위험물: IMDG 위험물 코드 입력 ② 냉동·냉장: 보관 온도(℃) 입력 ③ 고가품: 단가 및 총액 입력 ④ 중고품(Used/Secondhand): 중고품 여부 표시.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.1.2 오더 수정
**상세**: 등록된 오더의 정보를 수정. 수정 가능 범위: 송하인·수하인·화물 정보 전체. (비고: 오더가 Packing 되었거나 입고 이후 처리 단계에서는 수정 불가)
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.1.3 오더 삭제
**상세**: 오더 조회 목록에서 선택하여 삭제. 삭제 시 송하인·수하인·화물 정보 전체 삭제. (비고: Packing 또는 입고 이후 단계의 오더는 삭제 불가)
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.1.4 오더 조회
**상세**: 오더 목록 조회 및 상세 조회. 조회 조건: 회원명·법인명·부서명·등록일자. 상세 조회 시 오더정보·송하인·수하인·화물·서비스 정보를 모두 확인 가능.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

## 3.2 서비스 관리
**설명**: 오더에 운송 서비스(AIR·SEA·CIR·CCL)를 추가·수정·삭제하는 기능. 하나의 오더에 최대 4가지 서비스 추가 가능.
**Phase**: Phase 2

### 3.2.1 서비스 추가
**설명**: 오더에 운송 서비스를 추가. 서비스 구분코드(AIR/SEA/CIR/CCL)를 선택 후 세부 정보 입력. (비고: 최대 4개 서비스 등록 가능)
**Phase**: Phase 2

#### 3.2.1.1 AIR(항운) 서비스 추가
**상세**: 출발국가코드와 도착국가코드를 입력하여 항공운송수단 조회. 출발공항·도착공항·항공사·운항스케줄을 선택하면 서비스 등록 완료. 현재일 기준 운항 스케줄 자동 조회.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

#### 3.2.1.2 SEA(해운) 서비스 추가
**상세**: 출발국가코드와 도착국가코드를 입력하여 해운운송수단 조회. 출발항구·도착항구·선사·운항스케줄을 선택하면 서비스 등록 완료.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

#### 3.2.1.3 CIR(택배) 서비스 추가
**상세**: 도착국가코드로 서비스 가능한 택배사 목록 조회 후 선택. 택배사 선택 완료 시 운송장 출력 가능 상태로 전환.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

#### 3.2.1.4 CCL(통관) 서비스 추가
**상세**: 도착국가코드로 서비스 가능한 통관사 목록 조회 후 선택. 통관 서비스는 별도 통관신고 처리와 연동.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.2.2 서비스 수정
**상세**: 등록된 서비스의 운송수단·스케줄 정보를 수정. 서비스 구분코드(AIR/SEA/CIR/CCL)는 변경 불가하며 삭제 후 재등록 필요.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.2.3 서비스 삭제
**상세**: 오더에 등록된 서비스를 삭제. (비고: 입고 이후 처리 단계에서는 삭제 불가)
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.2.4 운항스케줄 조회
**상세**: AIR·SEA 서비스 추가 시 현재일 기준 운항 가능한 스케줄 목록 조회. 출발지·도착지·운송사 조건으로 필터링하여 원하는 스케줄 선택.
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

## 3.3 운송비용 관리
**설명**: 오더별 서비스 운송비용을 자동 계산하고 Extra Charge를 관리.
**Phase**: Phase 2

### 3.3.1 운송비용 자동 계산
**상세**: 화물 총중량·CBM 기반으로 운송원가 산출 후 영업이익율과 회원할인율 적용하여 운송비 자동 계산. 
(계산식: 운송비 = 원가 + (원가 × 영업이익율%) − (원가 × (1 − 할인율%)))
**권한**: 시스템
**Phase**: Phase 2

### 3.3.2 Extra Charge 입력
**상세**: 오더별 추가 비용(Extra Charge)을 수동 입력. 소수점 1자리까지 입력 가능(예: 15.5). (비고: 회원·운영자는 Extra Charge만 수정 가능, 관리자는 원가 포함 전체 수정 가능)
**권한**: 회원/운영자/관리자
**Phase**: Phase 2

### 3.3.3 운송비용 조회
**권한**: 회원/운영자/관리자
**Phase**: Phase 2
