# 7. 회계/청구
**설명**: 운송 서비스에 대한 청구서 발행·입금 확인·비용 관리·세금계산서 발행을 처리.
**Phase**: Phase 3

## 7.1 청구서 관리
**설명**: 오더·서비스별 운송비용을 집계하여 회원에게 청구서를 발행.
**Phase**: Phase 3

### 7.1.1 청구서 생성
**상세**: 회원별·기간별·운송수단별 조건으로 청구 대상 오더를 집계하여 청구서 생성. 청구서번호 자동 생성(예: INV-20260415-00001).
**정산 로직**:
1. **운송비**: `서비스 운송원가 + (원가 × 영업이익율%) - (원가 × 할인율%)`
2. **최종 청구액**: `∑(오더별 운송비) + ∑(오더별 Extra Charge)`
**권한**: 운영자/관리자
**Phase**: Phase 3

### 7.1.2 청구서 조회
**상세**: 청구서 목록 및 항목별 상세 내역 조회. 조회 조건: 회원명·기간·운송수단·청구상태(UNPAID/PARTIAL/PAID).
**권한**: 회원/운영자/관리자
**Phase**: Phase 3

### 7.1.3 청구서 출력
**상세**: 청구서를 PDF로 생성하여 출력 또는 다운로드. 청구서 내 항목별 운송비(원가/이익/할인금 상세표기 지원)·Extra Charge(소수점 1자리)·합계금액 포함.
**권한**: 회원/운영자/관리자
**Phase**: Phase 3

## 7.2 입금 관리
**설명**: 청구서별 입금 처리 및 미수금을 관리.
**Phase**: Phase 3

### 7.2.1 청구정보 생성
**상세**: 회원별·기간별·운송수단별 청구정보를 생성하여 입금 요청 기반 데이터 마련. 청구 생성 시 대상 오더의 운송비 합산.
**권한**: 운영자/관리자
**Phase**: Phase 3

### 7.2.2 입금 확인
**상세**: 청구서별 입금 여부를 확인하고 입금액·입금일시·입금방법을 등록. 부분 입금 시 청구상태를 PARTIAL로, 전액 입금 시 PAID로 변경.
**권한**: 운영자/관리자
**Phase**: Phase 3

## 7.3 수입 관리
**설명**: 회원별·기간별·운송수단별 수입 현황 조회. 운송비 합계·입금액·미수금을 집계하여 수익성 분석.
**권한**: 운영자/관리자
**Phase**: Phase 3

## 7.4 비용 관리
**설명**: 운송사별 실제 운송원가 조회. 수입 대비 원가를 비교하여 수익 분석.
**Phase**: Phase 3

### 7.4.1 항운(AIR) 비용 조회
**상세**: 항공사별·기간별 항운 원가(실제 지불 비용) 조회. 운송원가 마스터 기준으로 집계.
**권한**: 운영자/관리자
**Phase**: Phase 3

### 7.4.2 해운(SEA) 비용 조회
**상세**: 선사별·기간별 해운 원가 조회.
**권한**: 운영자/관리자
**Phase**: Phase 3

### 7.4.3 택배(CIR) 비용 조회
**상세**: 택배사별·기간별 택배 원가 조회.
**권한**: 운영자/관리자
**Phase**: Phase 3

## 7.5 세금계산서
**설명**: 청구서 기준 세금계산서 발행 및 조회.
**Phase**: Phase 3

### 7.5.1 세금계산서 조회
**상세**: 발행된 세금계산서 목록 조회. 세금계산서번호·발행일·공급가액·세액·합계금액 확인.
**권한**: 운영자/관리자
**Phase**: Phase 3

### 7.5.2 세금계산서 출력
**상세**: 세금계산서를 PDF로 생성하여 출력 또는 다운로드.
**권한**: 운영자/관리자
**Phase**: Phase 3

## 7.6 운송원가 관리
**설명**: 서비스·운송사·구간별 운송원가 기준 데이터를 등록·관리. 원가 데이터는 운송비 자동 계산의 기초 자료로 사용됨.
**Phase**: Phase 3

### 7.6.1 운송원가 등록
**상세**: 서비스구분(AIR/SEA/CIR)·운송사·출발지·도착지·중량구간별 단위원가 등록. 적용 시작일·종료일 및 기본 **영업이익률(%)**을 함께 설정하여 정산 엔진의 기초 데이터로 관리.
**권한**: 관리자
**Phase**: Phase 3

### 7.6.2 운송원가 수정
**권한**: 관리자
**Phase**: Phase 3
