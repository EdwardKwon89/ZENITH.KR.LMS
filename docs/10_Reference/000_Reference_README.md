## 📌 폴더 개요

이 폴더는 **깊이 있는 참조 문서**와 **정책 문서**를 포함합니다.

### 특징
- ✅ 완전한 규칙 및 표준
- ✅ 운영 정책 및 절차
- ✅ 고급 기술 가이드
- ✅ 프로젝트별 맞춤화 가능한 정책

### 사용 대상
- **처음 읽는 사람**: 너무 깊지 않음 (00_GUIDE 권장)
- **심화 학습**: 이 폴더에서 상세 정보 조회
- **PM/Leader**: 정책 문서 검토 및 커스터마이징

## 🎯 각 문서의 용도

### **001_Complete_Document_Writing_Guide.md**

**언제 읽을까?**
- 문서 작성에 대해 **완전히 이해**하고 싶을 때
- 새로운 폴더/문서 구조를 **설계**할 때
- 문서 표준을 **한 곳에서** 확인하고 싶을 때

**특징**
- 📖 약 11,500 토큰 (매우 상세)
- 🔍 모든 규칙과 예시 포함
- 📊 분류 시스템 완전 설명

**시작하기**
```
간단한 가이드: 00_GUIDE/001_Document_Writing_Guide.md
완전한 규칙: 10_Reference/001_Complete_Document_Writing_Guide.md
```

---

### **002_Audit_Management_Policy.md**

**언제 읽을까?**
- SAR 시스템의 **전사 정책**을 이해하고 싶을 때
- **여러 팀/프로젝트**에서 SAR을 표준화할 때
- Check List의 **조직 차원의 전략**을 세울 때

**특징**
- 📋 조직 차원의 정책
- 🔄 월간/분기별 운영 절차
- 📊 통계 및 리포팅 양식

## 📚 계층적 읽기 가이드

### **레벨 1: 입문 (초보자)**
```
1. 00_GUIDE/001_Document_Writing_Guide.md (핵심만)
2. 00_GUIDE/201_SAR_RULE.md (기본 개념)
3. 08_Self_Audit/000_Self_Audit_README.md (SAR 시스템)
```

```
1. 10_Reference/001_Complete_Document_Writing_Guide.md (완전 규칙)
2. 10_Reference/002_Audit_Management_Policy.md (정책)
3. 00_GUIDE/203_PROJECT_APPLICATION_CHECKLIST.md (실행)
```
소요 시간: 3-4시간

```
1. 모든 Reference 문서 정독
2. 00_GUIDE 전체 검토
3. 조직 정책 수립 및 커스터마이징
```
소요 시간: 6-8시간
