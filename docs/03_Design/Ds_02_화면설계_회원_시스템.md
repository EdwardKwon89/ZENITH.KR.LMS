> **작성자:** Antigravity (AI Agent)

> **버전:** v1.0

본 문서는 회원가입, 로그인, 계정 관리 및 시스템 관리 화면의 상세 설계를 정의합니다.

## 1. 인증 및 일반 사용자 화면

### 1.1 [SCR-001] 로그인 화면
시스템 접근을 위한 통합 로그인 게이트웨이입니다.
#### 1.1.1 레이아웃 및 구성
- 중앙 집중형 카드 레이아웃으로 구성됩니다.
- 입력부: 아이디(login_id), 비밀번호(password)
- 옵션: 아이디 저장(Checkbox), 자동 로그인(Checkbox)
#### 1.1.2 기능 로직 및 액션
- **로그인 버튼**: `An_01_데이터사전`의 비밀번호 검증 로직을 호출합니다.
- **소셜 로그인**: 카카오, 구글 API 연동 버튼을 하단에 배치합니다.
- **예외 처리**: 로그인 실패 5회 초과 시 '로봇 방지(CAPTCHA)'를 활성화합니다.

### 1.2 [SCR-004] 개인회원 가입 화면
신규 개인 사용자의 계정 생성을 지원합니다.
#### 1.2.1 가입 단계별 처리 (Wizard)
1. **약관 동의**: 필수/선택 항목 체크 및 문서 팝업 호출
2. **본인 인증**: 휴대폰 SMS 인증번호 발송 및 확인 (An_02 프로세스 1.1 참조)
3. **정보 입력**: 로그인 ID, PW, 이메일, 주소를 표준 입력 폼으로 구성합니다.
#### 1.2.2 자동화 정책
- 가입 완료와 동시에 회원 테이블에 **'iron'** 등급을 자동으로 매핑합니다.

## 2. 법인 및 관리자 전용 화면

### 2.1 [SCR-005] 법인회원 가입 신청 화면
B2B 고객의 가입 및 증빙 제출을 처리합니다.
#### 2.1.1 법인 특화 입력 필드
- 사업자등록번호(중복체크 필수), 법인명, 대표자명, 업태/업종
#### 2.1.2 파일 업로드 (Attachments)
- 사업자등록증 등 필수 증빙 파일을 드래그 앤 드롭으로 업로드합니다. (MinIO 연동)
#### 2.1.3 신청 결과 알림
- 신청 완료 시 "관리자 심사 중(PENDING)" 상태임을 명시적으로 안내합니다.

### 2.2 [SCR-006] 법인 가입 심사 대시보드 (Admin)
관리자가 신청된 법인을 검토하고 승인하는 화면입니다.
#### 2.2.1 심사 기능
- 업로드된 서류(MinIO)를 미리보기(Preview)로 확인합니다.
- **승인 액션**: 최종 승인 시 해당 법인에 **시스템 시리얼 6자리**를 ID로 부여합니다.
- **반려 액션**: 반려 사유를 입력하는 모달을 호출하며 시스템 알림을 트리거합니다.

## 3. 시스템 및 정보 관리 화면

### 3.1 [SCR-010] 내 정보 관리 화면
사용자가 등급, 활동 내역 및 정보를 확인하는 마이페이지입니다.
#### 3.1.1 대시보드 위젯
- 현재 등급(아이콘 표시), 최근 운송 횟수, 잔액(balance) 정보를 상단에 요약 노출합니다.
#### 3.1.2 정보 수정 액션
- 비밀번호 변경 및 전화번호/주소 업데이트가 가능하며, 변경 이력을 `updated_at`에 기록합니다.
