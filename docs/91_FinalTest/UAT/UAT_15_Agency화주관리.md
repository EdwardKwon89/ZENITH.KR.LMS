## [UAT-15-03] Agency 화주 상세 정보 및 등급/상태 수정

| 항목 | 내용 |
|:----|:----|
| 역할 | AGENCY (대리점) |
| 화면 URL | /ko/agency/shippers → /ko/agency/shippers/[id]/edit |
| 예상 소요 시간 | 5분 |
| 사전 조건 | 대리점 하위에 `BRONZE` 등급인 **개인(INDIVIDUAL) 화주**가 1개 이상 존재할 것 |

### 테스트 절차

| 순서 | 화면·URL | 수행 액션 | 입력 데이터 | 기대 결과 | 확인 |
|:---:|:---------|:---------|:-----------|:---------|:----:|
| 1 | /ko/agency/shippers | 수정할 개인 화주 행의 [상세 편집] 버튼 클릭 | — | 상세 편집 페이지(`/ko/agency/shippers/[id]/edit`)로 이동. 기존 등록 정보가 폼에 pre-fill됨. 개인 화주이므로 등급(grade) 선택박스가 활성화되어 표시됨 | ☐ |
| 2 | /ko/agency/shippers/[id]/edit | 등급을 `GOLD`로 변경 후 [저장] 클릭 | 등급: `GOLD` | "화주 정보가 수정되었습니다." 메시지 표시 및 목록 페이지로 리다이렉트 | ☐ |
| 3 | /ko/agency/shippers | 등급 변경 결과 목록 확인 | — | 해당 개인 화주 행의 등급이 `GOLD`로 표시됨 | ☐ |
| 4 | /ko/agency/shippers/[id]/edit | [상세 편집] 재진입 후 상태(isActive) 토글 스위치 클릭 후 [저장] | 상태 스위치 클릭 (활성 -> 비활성) | 스위치가 회색으로 변경되며 비활성 문구 표시됨. 저장 후 목록 페이지에서 해당 화주 상태가 `INACTIVE`로 변환됨을 확인 | ☐ |
| 5 | /ko/agency/shippers/[id]/edit | 연락처 수정 확인: [상세 편집] 재진입 후 연락처 변경 | 연락처: `02-9876-5432` | 자동 하이픈 포매팅 동작: `02-9876-5432` 형태로 실시간 변환됨. [저장] 후 목록 연락처 컬럼에 변경값 반영됨 | ☐ |
| 6 | Supabase Studio | `SELECT grade, is_active FROM zen_agency_shippers WHERE shipper_org_id = '[화주ID]'` | — | `grade` 컬럼이 `'GOLD'`, `is_active`가 `false`로 변경되었음을 확인 | ☐ |
