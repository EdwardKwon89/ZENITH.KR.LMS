## 현상 (What)

WBS 1.2.1 (기초 코드 테이블 설계 및 구축) 작업을 완료했다고 보고하고 체크리스트를 통과 처리했으나, 실제 프로젝트에는 `SERVICE_TYPE`, `INVOICE_STATUS` 등을 관리할 범용 공통 코드(Common Code) 테이블이 누락됨. 개별 마스터 테이블(`grade_master` 등)만 생성된 상태로 보고됨.

**발생 위치:** WBS 진행 상태 업데이트 및 체크리스트 로그 작성 시점

## 원인 (Why)

### 직접적 원인
"기초 코드"라는 용어의 비즈니스 범위를 자의적으로 해석하여, 일부 마스터 테이블 생성만으로 전체 작업을 완료한 것으로 오판함.

### 근본 원인
**검증 프로세스 부재**: 작업 완료 보고 전, 실제 파일 내용과 WBS 항목의 명세(`An_03` 등)를 1:1로 대조하는 최종 교차 검증 절차를 수행하지 않음.

### 기여 요소
- 텍스트 기반의 빠른 작업 진행으로 인한 "인지적 편향(Confirmation Bias)".
- 완료 체크리스트 작성 시 실제 코드 존재 여부를 기계적으로 체크함.

## 조치 (How)

### 수정 전
- `init_schema.sql`에 공통 코드 테이블 누락.
- `master_data.sql`에 공통 코드 데이터 누락.

### 수정 후
- `common_code_groups`, `common_codes` 테이블 추가 및 외래키 설정.
- `An_03`에 정의된 모든 공통 코드(MEMBER_STATUS, SERVICE_TYPE, INVOICE_STATUS) 데이터 적재 완료.

### 수정 범위
- [/] 스키마 수정 (진행 중 - Supabase 배포 예정)
- [/] 마스터 데이터 수정 (진행 중 - SQL 적재 예정)
- [x] WBS 상태 재점검 (0% 리셋 완료)
- [x] 점검 로그 업데이트 (CEO 감사 완료)

## 검증 (Verification)

### 테스트
사용자의 피드백을 통해 누락 사실을 인지하였으며, 수정 후 테이블 구조와 데이터가 `An_03_코드정의.md`와 일치함을 재확인함.
