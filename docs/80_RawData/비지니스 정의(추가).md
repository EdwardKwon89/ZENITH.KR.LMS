## 비즈니스 로직 확정 사항

### 운송비 계산
```
운송비 = 서비스운송원가 + (서비스운송원가 × 영업이익율(%)) - (서비스운송원가 × (1 - 회원별할인율(%)))
운송원가 = 화물정보의 총중량, 용적(CBM)으로 도출
Extra Charge = 오더별 수동 입력 (소수점 1자리)
```

### 법인 심사 프로세스
```
법인회원 가입
  → 사업자등록증 등 첨부파일 업로드
  → 관리자 첨부파일 확인
  → 승인 / 거부
  → 승인 시 법인 계정 활성화
```

### 개인회원 등급
- 특정 조건(발송횟수 등)에 따라 등급 부여
- 등급에 따른 운송비용 할인율 제공
