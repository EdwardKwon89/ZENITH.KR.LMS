## 🆘 PR 생성 후

### 리뷰 의견 받았을 때

```
1. 리뷰 의견 읽기
2. 수정 필요한 부분 파악
3. 로컬에서 수정
4. 다시 커밋: git commit -m "fix: 리뷰 의견 반영"
5. 푸시: git push
6. PR이 자동으로 업데이트됨
7. 완료될 때까지 반복
```

### 모든 리뷰를 받았을 때

```
PR 페이지에서:
[ ] "Merge pull request" 버튼 클릭
[ ] Confirm merge
[ ] "Delete branch" 클릭 (선택)

또는:
gh pr merge [PR_NUMBER]
```
