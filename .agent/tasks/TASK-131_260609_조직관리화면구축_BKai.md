4. i18n: ko/en/zh/ja 다국어 키 추가
5. NaviSidebar: Admin 섹션에 조직 관리 메뉴 항목 추가

**제외 범위**: CUSTOMS/DELIVERY 공개 회원가입 경로 구현 (별도 기획 필요, post-Go-Live)

   - `createOrganization(data)` — 신규 조직 생성 + optional 관리자 계정 생성

   - `updateOrganizationStatus(id, status)` — 상태 변경

2. Server Actions (`src/app/actions/admin/organization.ts`) 신규:
   - `createOrganizationAction(data)` — ACTIVE 상태로 즉시 생성 (Admin 직접 등록)
   - `getOrganizationsAdminAction(filter)` — 전체 조직 조회

경로: `src/app/[locale]/(dashboard)/admin/organizations/manage/`

**컴포넌트 구성**:
- `page.tsx` — Server Component: 초기 데이터 페칭
- `OrganizationManageClient.tsx` — Client Component: 목록 테이블 + 필터 + 등록 모달
- `OrganizationFormModal.tsx` — 신규 조직 등록 모달

**목록 테이블 컬럼**: 조직명, 유형(badge), 상태(badge), 사업자번호, 대표자, 생성일, 액션(상태변경)

**등록 필드**: 조직명, 유형(select), 사업자번호, 대표자명, 이메일, 전화번호
