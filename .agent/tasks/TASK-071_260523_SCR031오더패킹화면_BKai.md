### 결함 #2 (Critical) — Server Component onClick

`packing/page.tsx`는 `'use client'` 지시어 없는 서버 컴포넌트이나 L73·L88에 이벤트 핸들러(`onClick`) 사용. Next.js App Router에서 서버 컴포넌트는 이벤트 핸들러를 가질 수 없으며 **인쇄·뒤로가기 버튼이 브라우저에서 동작하지 않음**. 회귀 테스트가 신규 페이지를 커버하지 않아 미검출.

수정 방안: print/back 버튼 영역을 별도 `'use client'` 컴포넌트(`PackingActions.tsx`)로 분리.
