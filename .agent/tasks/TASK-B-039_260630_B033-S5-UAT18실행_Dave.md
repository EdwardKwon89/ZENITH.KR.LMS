### UAT-18-02 — UPS 발송 정보 RLS 격리 검증

**전제**: UAT-18-01 완료 (RELEASED 상태 + intl_ref_no 채번)

**흐름**:

| 순서 | 로그인 계정 | 액션 | 기대 결과 |
|:---:|:-----------|:----|:---------|
| 1 | UAT-18-TEST-001 소유 shipper 계정 | `/ko/orders` → 오더 상세 접근 | `intl_ref_no` 정상 표시 |
| 2 | **다른 shipper** 계정 | 동일 오더 URL 직접 접근 (`/ko/orders/[오더ID]`) | 접근 차단 또는 리다이렉트 |

> UAT-18-TEST-001 소유자 확인: Supabase Studio에서 `SELECT shipper_id FROM zen_orders WHERE id = '9ccc82e5-5814-4764-9f69-07c3e5100ec0'`  
> 다른 shipper 테스트 계정은 시스템에 등록된 타 SHIPPER 계정 사용 (예: agency_shipper@zenith.kr)

**DB 검증 SQL**:
```sql
-- RLS 격리 확인 (admin client 사용 시)
SELECT o.id, o.shipper_id, o.intl_ref_no, o.status
FROM zen_orders o
WHERE o.id = '9ccc82e5-5814-4764-9f69-07c3e5100ec0';
```
