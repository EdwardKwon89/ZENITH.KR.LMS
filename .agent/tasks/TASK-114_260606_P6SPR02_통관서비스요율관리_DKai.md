## 구현 명세

### 1. Server Actions (`src/app/actions/admin/customs-rates.ts`)

```typescript
interface CreateCustomsRateData {
  org_id: string;           // 통관사 조직 ID
  country_code: string;     // ISO 3166 (예: 'KR', 'US', 'JP')
  currency: string;         // 'USD' | 'KRW' | ...
  cost_per_kg?: number;
  cost_per_cbm?: number;
  fixed_fee?: number;
  transit_days?: number;
  valid_from: string;       // DATE
  valid_until?: string;     // DATE (null = 무기한)
}

createCustomsRate(data: CreateCustomsRateData): Promise<{ success: boolean; rate: CustomsRate }>
updateCustomsRate(id: string, data: Partial<CreateCustomsRateData>): Promise<{ success: boolean }>
getCustomsRates(orgId?: string): Promise<CustomsRate[]>
deleteCustomsRate(id: string): Promise<{ success: boolean }>  // soft delete: is_active = false
```

**권한 검증**:
- `createCustomsRate` / `updateCustomsRate` / `deleteCustomsRate`: ADMIN·MANAGER 또는 CUSTOMS_BROKER(본인 org_id 일치)
- `getCustomsRates`: authenticated (화주 조회 허용, is_active = true)

### 2. UI 페이지 (`src/app/[locale]/(dashboard)/admin/customs-rates/`)

**`page.tsx`** (서버 컴포넌트): 요율 목록 테이블
- ADMIN/MANAGER: 전체 통관사 요율 표시 + 조직 필터
- CUSTOMS_BROKER: 본인 org 요율만 표시
- 컬럼: 국가, 무게비용, 부피비용, 고정비용, 소요일, 유효기간, 버전, 상태(활성/만료)

**`CustomsRateFormModal.tsx`** (클라이언트 컴포넌트): 등록/수정 폼
- 국가 선택 (공통코드 또는 ISO 3166 드롭다운)
- 비용 입력 (무게당·부피당·고정)
- 유효기간 (valid_from / valid_until)
- 저장/취소

### 3. NaviSidebar 메뉴 추가
