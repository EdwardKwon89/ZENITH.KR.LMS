## 작업 범위

### 버그 원인
`OrderRegistrationForm.tsx`의 화주/수하인 탭 전환이 **조건부 렌더링(`&&`)** 으로 구현되어 있어, 탭 전환 시 DOM 언마운트 → 리마운트로 내부 `useState` 값이 초기화됨.

### 수정 방법
조건부 렌더링(`{infoTab === 'shipper' && (...)`) → **CSS `hidden` 클래스 토글**로 변경하여 DOM 유지.

```tsx
<div className={infoTab === 'shipper' ? '' : 'hidden'}>
  {/* 화주정보 탭 내용 */}
</div>
<div className={infoTab === 'consignee' ? '' : 'hidden'}>
  {/* 수하인정보 탭 내용 */}
</div>
```
