## [작업 범위]

### §1 — Server Actions: `src/lib/actions/ups-daily-close.ts`

| 함수 | 설명 |
|:-----|:----|
| `getDailyOutboundSummary(date: string)` | 당일(또는 지정일) UPS 출고 PKG 집계 — 건수·총중량·목적지별 분류 |
| `getDailyRevenueSummary(date: string)` | 당일 매출(selling_price 합계) / 매입(cost_price 합계) / 마진 집계 |
| `getDailyCloseHistory(from: string, to: string)` | 기간별 일마감 이력 조회 (날짜별 row) |

**조회 대상 테이블**:
- `zen_order_packages` — 출고 집계 (status IN ('OUTBOUND','SHIPPED'), shipped_at::date = :date)
- `zen_order_rate_snapshots` — 요율 스냅샷 (selling_price, cost_price)
- `zen_orders` — 오더 기본 정보 (shipper, dest_port 등)
- `zen_ups_base_rates` — Zone·product_type 참조

Zod 스키마:
```ts
export const DailyCloseDateSchema = z.object({
  date: z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
});
export const DailyCloseRangeSchema = z.object({
  from: z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
  to:   z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
});
```
