## 작업 범위

### §1 — `required-fields.tsx`: 유형별 할인율 분기 + readOnly prop 추가

**파일**: `src/app/[locale]/(dashboard)/agency/shippers/new/required-fields.tsx`

#### 1-A. 할인율 분기

```tsx
// Before: shipper_type + discount_rate 항상 grid-cols-2 병렬 표시
<div className="grid grid-cols-2 gap-4">
  <div>{/* shipper_type select */}</div>
  <div>{/* discount_rate input — 항상 표시 */}</div>
</div>

// After: INDIVIDUAL → discount_rate 숨김 (hidden value=0), CORPORATE → 표시
<div className="grid grid-cols-2 gap-4">
  <div>{/* shipper_type select */}</div>
  {shipperType === 'CORPORATE' && (
    <div>{/* discount_rate input */}</div>
  )}
</div>
{/* INDIVIDUAL일 때 discount_rate=0 hidden input */}
{shipperType === 'INDIVIDUAL' && (
  <input type="hidden" name="discount_rate" value="0" />
)}
```

#### 1-B. `readOnly` prop 추가 (상세보기용)

```tsx
interface RequiredFieldsProps {
  t: (key: string) => string;
  defaultValues?: Partial<{...}>;
  fieldErrors?: Record<string, string>;
  readOnly?: boolean;  // NEW: true 시 name·shipper_type·biz_no disabled
}
```

`readOnly=true` 적용 대상:
- `name` input → `disabled` + `className`에 `bg-slate-50` 추가
- `shipper_type` select → `disabled` + `bg-slate-50`
- `biz_no` input → `disabled` + `bg-slate-50` (CORPORATE인 경우)

`readOnly=true`여도 편집 가능한 필드:
- `discount_rate` (CORPORATE — 요율 변경 가능)
- `grade` (등급 변경 가능)
- `rep_name` (대표자명 변경 가능)
