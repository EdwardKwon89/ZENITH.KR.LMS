## 구현 명세

### 1. Server Actions (`src/app/actions/admin/delivery-rates.ts`)

```typescript
interface CreateDeliveryRateData {
  org_id: string;
  service_type: 'LOCAL' | 'TOTAL';
  // LOCAL 전용
  country_code?: string;
  // TOTAL 전용
  transport_mode?: 'AIR' | 'SEA';
  origin_code?: string;
  dest_code?: string;
  // 공통
  currency: string;
  cost_per_kg?: number;
  cost_per_cbm?: number;
  transit_days?: number;
  valid_from: string;
  valid_until?: string;
}

createDeliveryRate(data: CreateDeliveryRateData): Promise<{ success: boolean; rate: DeliveryRate }>
updateDeliveryRate(id: string, data: Partial<CreateDeliveryRateData>): Promise<{ success: boolean }>
getDeliveryRates(orgId?: string, serviceType?: 'LOCAL' | 'TOTAL'): Promise<DeliveryRate[]>
deleteDeliveryRate(id: string): Promise<{ success: boolean }>
```

**서버 측 검증**: service_type='LOCAL'이면 country_code 필수; 'TOTAL'이면 transport_mode·origin_code·dest_code 필수

**권한 검증**:
- CUD: ADMIN·MANAGER 또는 DELIVERY_AGENT(본인 org_id)
- R: authenticated (화주 조회 허용)

### 2. UI 페이지 (`src/app/[locale]/(dashboard)/admin/delivery-rates/`)

**`page.tsx`**: LOCAL / TOTAL 탭 분리 요율 목록
- LOCAL 탭: 국가·무게비용·부피비용·소요일·유효기간·상태
- TOTAL 탭: 운송수단·출발항·도착항·무게비용·부피비용·소요일·유효기간·상태

**`DeliveryRateFormModal.tsx`**: 등록/수정 폼
- service_type 선택 (LOCAL / TOTAL)
- 조건부 필드 표시 (LOCAL: 국가, TOTAL: 운송수단+노선)

### 3. NaviSidebar 메뉴 추가
