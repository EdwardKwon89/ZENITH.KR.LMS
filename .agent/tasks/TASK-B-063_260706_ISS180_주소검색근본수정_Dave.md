## 배경

TASK-B-062(PR#224)에서 CSP `script-src`에 `t1.kakaocdn.net`을 추가하여 스크립트 로드 차단은 해제됐으나, 로컬 테스트에서 주소 검색이 여전히 미동작함.

Jaison 재분석 결과 3가지 추가 문제 확인:

| # | 문제 | 심각도 |
|:-:|:-----|:------:|
| 1 | `useKakaoPostcodePopup` → `window.open()` Promise 내부 호출 → 브라우저 팝업 차단기에 의해 차단 | Critical |
| 2 | CSP `frame-src`에 `https://postcode.map.daum.net` 누락 (iframe 폴백 시 차단) | High |
| 3 | CSP `script-src`에 `https://t1.daumcdn.net` 누락 (추가 스크립트 로드 시 차단) | Medium |

## 작업 범위

### §1 — `address-input.tsx`: `useKakaoPostcodePopup` → `DaumPostcodeEmbed` 전환

**파일**: `src/app/[locale]/(dashboard)/agency/shippers/new/address-input.tsx`

popup window 방식은 브라우저 팝업 차단기에 취약합니다. `DaumPostcodeEmbed`를 인라인 모달로 표시하는 방식으로 전환합니다.

```tsx
// Before (팝업 방식 — 팝업 차단기 취약)
import { useKakaoPostcodePopup } from 'react-daum-postcode';
const openPopup = useKakaoPostcodePopup();
// onClick: openPopup({ onComplete: (data) => { ... } })

// After (인라인 임베드 방식)
import { DaumPostcodeEmbed } from 'react-daum-postcode';
// state: const [showPostcode, setShowPostcode] = useState(false);
// 버튼 onClick: setShowPostcode(true)
// JSX에 모달 추가:
```

```tsx
{showPostcode && (
  <div
    className="fixed inset-0 z-50 flex items-center justify-center bg-black/40"
    onClick={() => setShowPostcode(false)}
  >
    <div
      className="bg-white rounded-2xl overflow-hidden w-full max-w-md shadow-xl"
      onClick={(e) => e.stopPropagation()}
    >
      <DaumPostcodeEmbed
        onComplete={(data) => {
          setRoadAddress(data.roadAddress);
          setPostalCode(data.zonecode);
          setShowPostcode(false);
        }}
        style={{ height: 460 }}
      />
    </div>
  </div>
)}
```

버튼 코드 수정:
```tsx
// Before
<button type="button" onClick={handleKakaoSearch} ...>

// After
<button type="button" onClick={() => setShowPostcode(true)} ...>
```

`useKakaoPostcodePopup` import 및 `handleKakaoSearch` useCallback 제거.

### §2 — `next.config.ts`: CSP 보완

**파일**: `next.config.ts`

```ts
// 수정 전
"script-src 'self' 'unsafe-inline' 'unsafe-eval' https://t1.kakaocdn.net",
"img-src 'self' data: blob: https://t1.kakaocdn.net https://t1.daumcdn.net",

// 수정 후
"script-src 'self' 'unsafe-inline' 'unsafe-eval' https://t1.kakaocdn.net https://t1.daumcdn.net",
"img-src 'self' data: blob: https://t1.kakaocdn.net https://t1.daumcdn.net",
"frame-src 'none' https://postcode.map.daum.net",  // ← 신규 추가
```

> `DaumPostcodeEmbed`는 내부적으로 `https://postcode.map.daum.net`를 src로 하는 iframe을 생성합니다. `frame-src`가 없으면 `default-src 'self' blob:`으로 폴백되어 차단됩니다.

```ts
vi.mock('react-daum-postcode', () => ({

}));
```
