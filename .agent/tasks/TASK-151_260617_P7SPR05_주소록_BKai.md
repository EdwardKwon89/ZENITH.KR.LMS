## [작업 범위]

### §1. DB 스키마

- `zen_address_book` 테이블 신규
  - `org_id` / `user_id` 분기 소유 (CHECK 제약)
  - `display_mode`: `EN` | `BILINGUAL`
  - `is_default`, `recipient_name`, `recipient_address`, `recipient_address_local`, `recipient_phone`, `country_code`
- `zen_orders.recipient_address_local` 컬럼 추가
- `create_order_atomic` RPC에 `recipient_address_local` 파라미터 추가

### §2. Server Actions 4종

- `getAddressBook` — 주소록 목록 조회
- `createAddressBook` — 주소록 등록
- `updateAddressBook` — 주소록 수정
- `deleteAddressBook` — 주소록 삭제

Zod 스키마 기반 입력 검증 포함.

### §3. 주소록 관리 페이지

- `/[locale]/(dashboard)/address-book/page.tsx`
- `AddressBookClient.tsx` — 목록, 등록/수정/삭제, 기본 주소 설정
- `AddressBookSelector.tsx` — 주소 선택용 콤보 컴포넌트

### §4. OrderRegistrationForm 연동

- 수취인 주소 입력 영역에 "주소록에서 선택" 기능 추가
- 선택 시 `recipient_name`, `recipient_address`, `recipient_address_local`, `recipient_phone`, `recipient_zipcode`, `recipient_pccc`, `recipient_email` 자동 채움

### §5. NaviSidebar 메뉴

- ADMIN/MANAGER 역할에게 "주소록" 메뉴 표시
- i18n 키 추가 (`ko.json`, `en.json`)
