## [목표]

UPS 국제 특송 오더 등록 시 자주 사용하는 수하인 주소를 저장·재사용할 수 있는 주소록 기능을 구현한다.

**핵심 흐름**:
1. 주소록 관리 페이지: 등록 / 수정 / 폐기(soft delete)
2. 오더 등록 폼 연동: "주소록에서 선택" 버튼 → 모달 → 선택 → 수하인 필드 자동 입력
3. 표기 방식 선택: EN 단독 / BILINGUAL (현지어+영어 병기)

## [설계 확정] — Edward 승인 2026-06-17

### 소유권 범위
- **org 소속 사용자**: `org_id` SET → 조직 내 공유
- **개인 사용자** (org 없음): `org_id` NULL, `created_by` SET → 본인만 접근

```sql
CONSTRAINT chk_owner CHECK (org_id IS NOT NULL OR created_by IS NOT NULL)
```

### 주소 표기 방식 (`display_mode`)
- `EN`: 영문 주소만 사용 (기본값)
- `BILINGUAL`: 현지어 주소 + 영문 주소 병기 (오더 폼에 두 필드 노출)

## [작업 범위]

### §1 — Migration: `zen_address_book` 테이블 신설

```sql
CREATE TABLE public.zen_address_book (
  id                    UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  org_id                UUID REFERENCES zen_organizations(id) ON DELETE CASCADE,
  created_by            UUID NOT NULL REFERENCES zen_profiles(id),

  label                 TEXT NOT NULL,              -- 별칭 (예: "Tokyo Warehouse")

  -- 수하인
  consignee_name        TEXT NOT NULL,              -- 영문 (필수)
  consignee_name_local  TEXT,                       -- 현지어 (BILINGUAL 시)

  -- 영문 주소 (UPS 필수)
  address1_en           TEXT NOT NULL,
  address2_en           TEXT,
  city_en               TEXT NOT NULL,
  state_en              TEXT,
  postal_code           TEXT NOT NULL,
  country_code          CHAR(2) NOT NULL,           -- ISO 3166-1 alpha-2

  -- 현지어 주소 (BILINGUAL 선택)
  address1_local        TEXT,
  address2_local        TEXT,
  city_local            TEXT,
  state_local           TEXT,

  -- 표기 방식
  display_mode          TEXT NOT NULL DEFAULT 'EN'
                        CHECK (display_mode IN ('EN', 'BILINGUAL')),

  phone                 TEXT,
  email                 TEXT,
  is_active             BOOLEAN NOT NULL DEFAULT TRUE,
  created_at            TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at            TIMESTAMPTZ NOT NULL DEFAULT NOW(),

  CONSTRAINT chk_owner CHECK (org_id IS NOT NULL OR created_by IS NOT NULL)
);

-- RLS
ALTER TABLE public.zen_address_book ENABLE ROW LEVEL SECURITY;

-- org 사용자: 같은 조직 전체 접근
CREATE POLICY "addr_book_org_select" ON public.zen_address_book
  FOR SELECT USING (
    org_id IS NOT NULL AND
    org_id IN (SELECT org_id FROM zen_profiles WHERE id = auth.uid())
  );

-- 개인 사용자: 본인 소유만 접근
CREATE POLICY "addr_book_personal_select" ON public.zen_address_book
  FOR SELECT USING (
    org_id IS NULL AND created_by = auth.uid()
  );

-- Insert/Update/Delete: 소유자 본인만
CREATE POLICY "addr_book_insert" ON public.zen_address_book
  FOR INSERT WITH CHECK (created_by = auth.uid());

CREATE POLICY "addr_book_update" ON public.zen_address_book
  FOR UPDATE USING (created_by = auth.uid());

CREATE POLICY "addr_book_delete" ON public.zen_address_book
  FOR DELETE USING (created_by = auth.uid());
```
