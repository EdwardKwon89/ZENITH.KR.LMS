| 단계 유형 | 내용 |

| 각 전이마다 2단계 | ① 상태 배지 클릭 → `{대상상태}` 선택 → 변경 → 토스트 확인 |
| | ② 배지 색상이 `ORDER_STATUS_META` 정의와 일치 확인 |
| 전이 목록 (7개) | REGISTERED→SCHEDULED(남색) → WAREHOUSED(노랑) → PACKED(주황) → RELEASED(보라) → IN_TRANSIT(청록) → DELIVERED(초록) |
| 마지막 단계 | 오더 상세(`/ko/orders/[id]`) → 히스토리 탭 → **7개 전이 이력 전체** 기록 확인 (변경 전·후·일시·수행자) |

- 전 단계 ☑ 완료
- REGISTERED→SCHEDULED→WAREHOUSED→PACKED→RELEASED→IN_TRANSIT→DELIVERED 7개 전이 각각 성공
- 각 전이 후 상태 배지 색상이 ORDER_STATUS_META 정의와 일치
