### §2 — `src/lib/validations/agency.ts`

`CreateAgencyShipperSchema`에 필드 추가 및 CORPORATE 조건부 필수 검증:

```typescript
export const CreateAgencyShipperSchema = z.object({
  name: z.string().min(1).max(100),
  shipper_type: z.enum(['INDIVIDUAL', 'CORPORATE']),
  discount_rate: z.number().min(0).max(0.9999),
  grade: z.string().max(20).optional(),
  biz_no: z.string().min(1).max(20).optional(),
  rep_name: z.string().max(50).optional(),
  contact_name: z.string().optional(),
  contact_email: z.string().email().optional(),
  contact_phone: z.string().optional(),
}).superRefine((data, ctx) => {
  if (data.shipper_type === 'CORPORATE' && !data.biz_no) {
    ctx.addIssue({
      code: z.ZodIssueCode.custom,
      message: '법인 화주는 사업자번호가 필수입니다.',
      path: ['biz_no'],
    });
  }
});
```

### §3 — `src/app/actions/agency/shippers.ts`

`_createShipperOrg` 함수 시그니처 확장 및 INSERT에 `biz_no`·`rep_name` 포함:

```typescript
async function _createShipperOrg(
  supabase: SupabaseClient,
  name: string,
  biz_no?: string,
  rep_name?: string,
): Promise<string> {
  const { data, error } = await supabase
    .from('zen_organizations')
    .insert({ name, type: 'SHIPPER', status: 'ACTIVE', biz_no: biz_no ?? null, rep_name: rep_name ?? null })
    .select('id')
    .single();
  if (error) throw new Error(`Failed to create shipper org: ${error.message}`);
  return data.id;
}
```

`createAgencyShipper`에서 호출 시 `parsed.data.biz_no`, `parsed.data.rep_name` 전달.
