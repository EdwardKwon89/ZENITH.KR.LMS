## 배경

신규 오더 등록 첫 화면에서 부피·무게 입력 후 표시되는 "Estimated Freight"가 시스템에 등록된 현실적인 요율 데이터가 아닌 **fallback dummy rate** ($5.5/kg AIR, $9/kg EXP, $120/CBM SEA)로 계산되어 실제 운임과 괴리가 있음.

실제 billing은 DB의 `zen_rate_cards`/`zen_surcharges` 기반 `calculateCompositePricing()`에서 처리되므로, **현실적인 시드 데이터를 등록**하여 demo/development 환경에서도 실제와 유사한 운임이 제시되도록 함.

## 수정 내용

### 신규 파일: `supabase/seed_rates_realistic.sql`

**운송사 7개 추가** (기존 ZENITH_AIR/SEA 포함 총 9개):

| Carrier | Mode | 주요 Route |
|---------|------|-----------|
| Korean Air Cargo | AIR | ICN→LAX $3.50/kg~, ICN→NRT, ICN→FRA |
| Asiana Cargo | AIR | ICN→LAX $3.30/kg~ |
| FedEx Express | EXP | ICN→LAX $12.00/kg~, ICN→SIN |
| DHL Express | EXP | ICN→LAX $11.50/kg~ |
| HMM | SEA | PUS→LGB $65/CBM~ |
| MSC | SEA | PUS→LGB $60/CBM~, PUS→SIN |
| Evergreen | SEA | PUS→LGB $62/CBM~ |

**Rate Cards 12건** — slab tier 기반 (45/100/300/500kg 임계값):

```json
{
  "weight_slabs": [
    {"weight_min": 0, "unit_price": 3.50, "min_charge": 75},
    {"weight_min": 45, "unit_price": 2.80, "min_charge": 126},
    {"weight_min": 100, "unit_price": 2.50, "min_charge": 250}
  ],
  "cbm_slabs": [
    {"cbm_min": 0, "cbm_price": 584.50, "min_charge": 75}
  ]
}
```

**Surcharges 30건** — carrier-level (FSC 25-28%, SSC $10-12 flat, THC $0.30-0.35/kg, BAF 16-20%)

**Route Network** — 모든 rate card의 port 조합에 대해 `zen_route_network` 자동 생성

### 속성별 3계층 구조 적용

| 항목 | 예시 (KAL ICN→LAX 150kg) |
|:-----|:--------------------------|
| **carrier_cost** | $1.50/kg (운송사 원가) |
| **margin_rate** | 15.0% (운송사 이윤율) |
| **platform_fee_rate** | 5.0% (플랫폼 수수료율) |
| **→ unit_price (산출)** | $2.50/kg (150kg slab) |
| **→ baseFreight** | 150kg × $2.50 = $375.00 |
| **→ surcharges** | FSC 28%=$105 + SSC $12 + THC $0.35/kg=$52.50 |
| **→ Total** | **$544.50** |

## 적용

```bash
psql "postgresql://postgres:postgres@127.0.0.1:54322/postgres" \
  -f supabase/seed_rates_realistic.sql
```

- Idempotent: `ON CONFLICT DO NOTHING` + `NOT EXISTS` 조건 사용
- 로컬 Supabase DB 기준 실제 컬럼(`zen_rate_cards`에 없는 `transit_days`/`origin_code`/`dest_code` 등 제외)에 맞게 조정 완료
- DB 적용 후 9개 운송사, 12건 rate card, 30건 surcharge 등록 확인 완료

## 잔여 이슈

UI "Estimated Freight"는 여전히 `freight-calculator.ts`의 fallback dummy rate ($5.5/kg AIR)로 계산됨.
실제 billing은 DB rate로 정상 처리되며, UI 추정치 현실화는 별도 IMP로 등록 가능.
