## 발견 경위

UAT 화주목록(`/agency/shippers`) 화면 확인 중 발견.  
TASK-B-048(DEF-091)에서 "상세 편집" 기능이 추가된 이후에도 인라인 수정 버튼이 잔존.

## 현상

**Defect A: 등록일 다음 컬럼(Actions) 잘림**
- `actions` 컬럼에 "상세 편집" + "수정" 버튼 2개가 나란히 표시됨
- 테이블 `min-w-[720px]` 기준 너비 부족으로 마지막 컬럼이 시각적으로 잘림

**Defect B: 인라인 "수정" 버튼 불필요**
- 상세편집 페이지(`/edit`) 추가로 인라인 등급/할인율 수정 기능이 중복됨
- "수정" 버튼 클릭 시 등급 드롭다운·할인율 입력으로 전환되는 인라인 편집 로직 잔존
- 해당 기능은 `/edit` 페이지로 통합 가능 → 제거 대상

## 영향 범위

- `src/app/[locale]/(dashboard)/agency/shippers/editable-cell.tsx` — ActionCell 수정 버튼, EditableGradeCell, EditableRateCell
- `src/app/[locale]/(dashboard)/agency/shippers/shipper-table-row.tsx` — 인라인 편집 props
- `src/app/[locale]/(dashboard)/agency/shippers/shipper-table.tsx` — 인라인 편집 props
- `src/app/[locale]/(dashboard)/agency/shippers/shippers-client.tsx` — 인라인 편집 state/handler

## 권장 조치

| 파일 | 조치 |
|:-----|:-----|
| `editable-cell.tsx` | ActionCell에서 "수정" 버튼 제거. EditableGradeCell·EditableRateCell을 표시 전용으로 단순화 |
| `shipper-table-row.tsx` | isEditing·onEdit·onCancel·onSave·editGrade·editRate 등 인라인 편집 props 제거 |
| `shipper-table.tsx` | 동일 props 제거 |
| `shippers-client.tsx` | editingId·editGrade·editRate state, startEdit·cancelEdit·saveEdit handler 제거 |
