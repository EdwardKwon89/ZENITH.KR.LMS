# Project Roadmap: ZENITH_LMS

## Phase 1: Foundation & Core Identity (v2.1.0 Premium Governance)
> **목표**: 시스템 인프라 구축 및 자기 진화형 품질 거버넌스 체계 확립 [Status: ✅ Phase 1 Stabilized]
- [x] 개발 환경 설정 및 Next.js 프로젝트 초기화
- [x] **[01. 로그인/인증]** 기능 구현 (Premium UI 업그레이드 및 Redirect Guard 반영 완료)
- [x] **[02. 회원관리]** (법인 회원 가입 및 관리자 승인 시스템 구축 완료)
- [x] **[09. 시스템관리(기초)]** (관리자 전용 거버넌스 설정 UI 및 정책 연동 완료)
- [x] **[11. 요율 거버넌스(TISA)]** (요율 스냅샷 및 TISA 규칙 엔진 반영 완료)
- [x] **[Asset] 지능형 품질 인프라** (Phase 1~3 LIVE Checklist 및 SAR 자산화 완료)
- [x] **[12. 디자인 시스템]** (Premium Glass Component 및 고밀도 데이터 그리드 표준 확립 완료)
- **Milestone**: 전 공정 품질 거버넌스 체계 완비 및 브랜드 아이덴티티(Design System) 확립 단계 [100% 완료 - **Stabilized**]

## Phase 2: Core Logistics Operations (v1.1.0)
> **목표**: 실제 물류 비즈니스의 핵심인 오더 및 창고 관리 기능 구축 [Status: ✅ Operating stabilized]
- [x] **[09. 시스템관리(고도화)]** (Phase 2.1 마스터 데이터 및 어드민 보안 시스템 구축 완료)
- [x] **[Governance] Proxy/Middleware 수복** (루프 방지 및 다국어 정합성 확보 완료)
- [x] **[Security] 전사적 오류 처리 및 가딩 고도화** (Global Error Boundary & ZenErrorView 반영 완료)
- [x] **[03. 오더관리]** (B2B/B2C 통합 오더 접수 고도화, 실시간 예상 운임 산출 및 인프라 동적 필터링 반영 완료)
- [x] **[04. 마스터오더관리]** (오더 그룹핑, 배치 생성 및 하우스 오더 래핑 고도화 완료)
- [x] **[05. 창고관리]** (바코드 기반 입출고 제어 및 GS1-128 국제 표준 바코드 시스템 구축 완료)
- **Milestone**: 지능형 오더 관제 및 거버넌스 대시보드 가동 [85% 진행 중 - 주요 핵심 로직 완료]

## Phase 3: Global Tracking & Settlement (v1.2.0)
> **목표**: 외부 시스템 연계(Tracking/통관) 및 비즈니스 수익 창출을 위한 회계 기능
- [ ] **[06. 운송/Tracking]** (실시간 운송 상태 추적 및 위치 API 연계)
- [ ] **[07. 회계/청구]** (청구서 자동 발행, 입금 처리 및 수익성 분석)
- [ ] **[08. VOC 관리]** (오더별 고객 불만 및 문의 자동 관리)
- **Milestone**: 실시간 트래킹 데이터 시각화 및 청구-입금 완료 프로세스 검증

## Phase 4: Customer Support & Intelligence (v1.3.0)
> **목표**: 고객 편의성 강화 및 통계 기반의 경영 의사결정 지원
- [ ] **[09. 시스템관리(고도화)]** (운송/비용 통계 차트 및 대시보드)
- [ ] **[10. 고객지원]** (FAQ, 공지사항, 1:1 상담 게시판)
- [ ] 시스템 성능 최적화 및 자동 백업/복구 시스템 검증
- **Milestone**: 전 사적 물동량 통계 리포트 생성 및 운영 프로세스 최적화 완료
